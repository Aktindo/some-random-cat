# Random

## Methods

#### `.newCat()`  `STATIC`

Random method to generate a random cat image.

| Type | Description |
| :--- | :--- |
| any | URL for the image. |

{% hint style="success" %}
Returns: [`Promise<Pending>`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise)\`\`
{% endhint %}
